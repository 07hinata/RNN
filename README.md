# Projekt – Analýza časových řad s využitím rekurentních neuronových sítí
Tento repozitář obsahuje data a zdrojové kódy použité pro bakalářskou práci zaměřenou na analýzu časových řad s využitím rekurentních neuronových sítí (RNN).

## Struktura projektu
- `cisteni_dat.ipynb` - příprava a čištění dat
- `dly-0-203-0-11651-T.csv` - data použitá pro modely
- `modely.pdf` - export výsledků a vizualizace z notebooku `modelyF.ipynb`
- `modelyC.ipynb` - modely (bez outputů)
- `modelyF.ipynb` - modely (s outputy)
- `all_data.zip` - archiv obsahující všechna data použitá v projektu

## Poznámky
- Zdroj dat `dly-0-203-0-11651-T.csv` je Český hydrometeorologický ústav (CHMI). Soubor byl stažen z webových stránek: https://opendata.chmi.cz/meteorology/climate/historical/data/daily/. Soubor nebyl přejmenován pro lepší dohledatelnost původního souboru.
- Soubor `modely.pdf` není v náhledu GitHubu správně zobrazen. Pro správné zobrazení jej lze stáhnout.
- Notebook `modelyC.ipynb` neobsahuje výstupy z důvodu kompatibility s GitHubem. Vizualizace a výsledky jsou dostupné v souboru `modely.pdf`.
- Notebook `modelyF.ipynb` obsahuje výstupy, ale z důvodu kompatibility s GitHubem je pro lepší vizualizaci použit soubor `modely.pdf`. Avšak tento notebook je nahrán pro úplnost dat.
